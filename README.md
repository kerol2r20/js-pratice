## 使用方法
1. clone 這個方案
2. yarn 安裝依賴模組
3. `yarn gendata` 產生測試檔案
4. 實現 `src/helper.js`

## 實作說明
主要實作 3 個 function，`getNames`、`filterNormalBMI`、`genderAvgWeightAndHeight`，在檔案中的註解有解釋實作的目的。
